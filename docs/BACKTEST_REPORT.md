## How to Generate This Report

Run the full backtest pipeline:

```bash
# 1. Run Monte Carlo backtest on 2025 season (sample 30 days for speed)
python scripts/backtest_simulator.py \
    --start 2025-04-01 --end 2025-09-30 \
    --sims 5000 --prop-types K,TB,H,HR \
    --sample-days 30 \
    --output-dir output/backtest \
    --seed 42

# 2. Generate this report with charts
python scripts/generate_backtest_report.py \
    --predictions output/backtest/backtest_predictions_2025-04-01_2025-09-30.json \
    --summary output/backtest/backtest_summary_2025-04-01_2025-09-30.json \
    --daily output/backtest/backtest_daily_2025-04-01_2025-09-30.json \
    --output-dir output/backtest \
    --report-path docs/BACKTEST_REPORT.md

# 3. (Optional) Run model comparison to quantify factor values
python scripts/compare_models.py \
    --start 2025-07-01 --end 2025-07-31 \
    --configs baseline,no_umpire,career_only,point_estimate \
    --prop-types K \
    --sample-days 10 \
    --output-dir output/backtest/compare \
    --generate-report

# 4. (Optional) Generate demo report with synthetic data
python scripts/generate_backtest_report.py --demo \
    --report-path docs/BACKTEST_REPORT.md
```

## System Architecture

```
scripts/backtest_simulator.py    → Monte Carlo engine (5,000 sims/game)
scripts/generate_backtest_report.py → Charts + Markdown report
scripts/compare_models.py        → Multi-model A/B testing
configs/full_model.json          → Full model configuration
configs/career_only.json         → Minimal v1.0-equivalent config
```

### Prop Types Supported
- **K** — Pitcher strikeouts (per-PA K rate × batters faced × innings)
- **TB** — Batter total bases (multinomial PA outcomes: 1B/2B/3B/HR)
- **H** — Batter hits (same model, tracking hits only)
- **HR** — Batter home runs (same model, tracking HRs only)

### Confidence Tiers
- **A** — Best-side P(over/under) > 65%
- **B** — Best-side P(over/under) 60–65%
- **C** — Best-side P(over/under) 55–60%
- **D** — Best-side P(over/under) < 55%

### Betting Methodology
- Quarter Kelly (25%) on edges > 3%
- Max 5% of bankroll per bet
- Standard -110 juice assumed
- Track CLV (Closing Line Value) when available
